# Roam2Github

This project was inspired by by [MatthieuBizien/roam-to-git](https://github.com/MatthieuBizien/roam-to-git) and [everruler12/roam2github](https://github.com/everruler12/roam2github)

Roam-to-git has offered me great peace of mind knowing my Roam data is safe. However, my backups regularly failed with unknown errors multiple times a week. People were emailing me with the same issues, and I couldn't help. Then it got to the point on 2021-01-28 where all my backups were failing. Roam-to-git's creator didn't seem active with addressing the issues, and I don't know enough Python fix his code. So I decided to write my own backup solution from scratch using Nodeâ€” with clearer logging to make troubleshooting easier.
